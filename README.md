# fatty-suite — Command Centre

Control surface for the Fatty autonomous development system. Manage the agents,
plan work, and observe state from here.

Start with **`CLAUDE.md`** for the operator guide. Role/polling/model docs are in
**`docs/`**, and the management skills + planner subagent are in **`.claude/`**.

The product repo (`fatty/`) and the three agent repos are independent git repos,
gitignored here — this repo tracks only the command-centre files.
